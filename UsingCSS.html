Using CSS

This page contains some text that we can use to work with styles and change the way some of the elements appear on the page.

  • Styles give us consistency on the page
  • Styles make it fast and easy to change the look of a page
  • Styles are powerful and have many features that can considerably improve the look of a page

Creating a style for a built in tag:

  1. Specify the name of the tag
  2. Add each style element and its value to the list
  3. Multiple elements are separated by a ";" and usually placed on separate lines to increase readability

Creating a style for a named section of the page:

  1. Identify the id for that block of text
  2. Specify the id of the of the text using a "#" before the id
  3. Add each style element and its value to the list
  4. Multiple elements are separated by a ";" and usually placed on separate lines to increase readability
